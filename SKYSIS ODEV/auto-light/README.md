## Auto Light System
In this projects, light turn on when the room is dark.
### Components

|Quantity|Name|
|--------|----|
|1|Ardunio Uno R3|
|1|Light Bulb|
|1|200 ohm resistor|
|1|Ambient Light Sensor[Phototransistor]|
